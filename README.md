# Pomodoro
Pomodoro timer application with additional features.

## Technologies
- (https://rust-lang.org/)[Rust]
- (https://ratatui.rs/)[Ratatui]

## Run and build
Build the application with:
```sh
cargo build
```
Run the application with:
```sh
cargo run
```
