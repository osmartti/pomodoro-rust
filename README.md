# Pomodoro
Pomodoro timer application with additional features.

## Technologies
- [Rust](https://rust-lang.org/)
- [Ratatui](https://ratatui.rs/)

## Run and build
Build the application with:
```sh
cargo build
```
Run the application with:
```sh
cargo run
```
